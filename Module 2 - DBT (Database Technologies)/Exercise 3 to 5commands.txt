10. What is the difference between COUNT(Status) and COUNT(*) ? 
-->
Count(Status): count the rows using rows of particular column.
Count(*): counts the rows using the rows of entire table.

 
11. Display the Status and the Count of Suppliers with that Status in the following format 
as shown below:- 
 
Status Count 
Ten 1 
Twenty 2 
Thirty 3
-->
select
case
when status=10 then 'Ten'
when status=20 then 'Twenty'
when status=30 then 'Thirty'
end "Status",

from s group by status order by 2;
